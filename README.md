# 💰 Digital Wallet Tracker

A simple and efficient **digital wallet application** that helps users record and manage their daily transactions digitally — eliminating the need for traditional pen-and-paper methods.

---

## 🚀 Features

* 💵 Add, edit, and delete transactions
* 📊 Automatic balance calculation
* 📂 Categorize expenses (Food, Travel, Bills, etc.)
* 📅 View transaction history
* ⚡ Fast and user-friendly interface
* 📱 Mobile-friendly design

---

## 🎯 Project Objective

The goal of this project is to **digitize personal finance tracking** by providing a simple, reliable, and accessible platform where users can manage their income and expenses anytime, anywhere.

---

## 🛠️ Tech Stack

* **Backend:** ( Node.js )
